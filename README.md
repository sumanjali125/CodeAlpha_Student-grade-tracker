# Student Marks Analyzer 🎓

A simple Java console program to analyze marks of multiple students and generate a summary report.

### ✨ Features
- Input marks for N number of students
- Uses `ArrayList` to store and manage student marks dynamically
- Calculates **Total Students**, **Average Marks**, **Highest Marks**, and **Lowest Marks**
- Displays a clean summary report in console
- Handles dynamic input using Scanner class

### 🛠️ Tech Stack
- **Language**: Java
- **Concepts Used**: ArrayList, Loops, Conditional Statements, Type Casting
- **IDE**: VS Code / IntelliJ IDEA / Any Java Compiler
